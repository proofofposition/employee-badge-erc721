# POPP NFT

This is a smart contract that represents the POPP job `badge` to verify your position in a company.

```shell
npx hardhat help
npx hardhat test
mv .env.example .env
GAS_REPORT=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.js
```
